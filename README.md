# Brewery Bees 

Projeto desenvolvido em equipe durante o Bootcamp Mobile iOS da CI&T. 

O objetivo do app é conseguir pesquisar cervejarias na região, visualzar seus detalhes, avaliar e salvar em uma lista de favoritos.
